# Machine-learning-the-Ising-phase-transition

These scripts are for analyzing the critical behavior of the 2D Ising model in various geometries and symmetries using machine learning tools. 
First we sample the partition function of the 2D Ising model using Monte Carlo simulations. Next we use convolutional neural networks trained on thermally equilliberated configurations to predict critical temperatures and scaling exponents. Unsupervised learning techniques such as principle component analysis (PCA) are used to determine order parameters. 
This project is inspired by the results in Carrasquilla, J., & Melko, R. G. (2017). Machine learning phases of matter. Nature Physics, 13(5), 431–434. https://doi.org/10.1038/nphys4035
